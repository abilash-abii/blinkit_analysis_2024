## Overview

A Power BI dashboard analyzing real-time grocery retail data — built to visualize sales trends and inventory levels for a quick-commerce retail context (modeled on Blinkit's grocery delivery business).

## Business Problem

Retail and grocery operations need continuous visibility into sales trends and stock levels to avoid overstocking or running out of high-demand items, and to spot which products/categories are driving customer engagement.

## Approach

- Imported and modeled grocery sales and inventory data in Power BI
- Built interactive visuals tracking sales trends over time
- Built inventory-level views to flag stock risk by category
- Structured the report to support both a high-level summary view and category-level drill-down

## Tech Stack

Power BI · DAX · Power Query

## Key Insights

- Sales trends vary meaningfully by category, which has direct implications for inventory planning
- Inventory level tracking surfaces categories at risk of stockout or overstock before they become a problem

- Publish the report to the web via Power BI's "Publish to web" feature and link it here, **or**
